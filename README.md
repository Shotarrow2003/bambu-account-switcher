# Bambu Studio — Mac で複数アカウントを切り替える方法

Bambu Studio を **複数アカウントでワンクリック切り替え**できるようにするセットアップ手順です。
アカウントは2個以上でも自由に追加できます。ただし同時に起動し画面を見ることはできません。

---

## 方式：シムリンク入れ替え方式

Bambu Studio はデータを以下に**固定で**保存します：

```
~/Library/Application Support/BambuStudio/
```

このフォルダ自体をシムリンクにして、起動時に差し替えます。

```
~/Library/Application Support/
  BambuStudio              ← シムリンク（起動時に切り替わる）
  BambuStudio_家/          ← 家アカウントの実データ
  BambuStudio_会社/        ← 会社アカウントの実データ
  BambuStudio_サブ/        ← 追加アカウントも自由に作れる
```

---

## セットアップ手順

### 1. このリポジトリをダウンロード

```bash
git clone https://github.com/Shotarrow2003/bambu-account-switcher.git
cd bambu-account-switcher
```

### 2. セットアップスクリプトを実行

```bash
bash setup.sh
```

登録名を聞かれるので入力します（例: `家`）。
実行すると：
- 現在のログイン済みデータ → 入力した名前のアカウントとして保存
- デスクトップに起動ショートカットを生成

### 3. アカウントを追加

```bash
bash add_account.sh
```

アカウント名を入力するとデータフォルダとデスクトップショートカットが作成されます。
何個でも追加できます。

### 4. 完了

デスクトップに `アカウント名_BambuStudio.app` が出現します。
ダブルクリックするだけで切り替えて起動できます。

**初回のみ**、各アカウントのショートカットから起動してログインしてください。次回以降はログイン不要です。

---

## 注意事項

- 同時起動はできません（切り替え時は既存のBambu Studioが自動終了します）
- `/Applications/BambuStudio.app` がインストール済みであることが前提です
- アカウントのデータは削除されません。シムリンクの切り替えのみ行います

---

## アンインストール

```bash
bash uninstall.sh
```

実行すると：
- シムリンクを元の実フォルダに戻す
- 残りのアカウントデータの削除確認（選択式）
- デスクトップのショートカットをすべて削除
